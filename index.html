GIMPshop Reloaded

GIMPshop Reloaded is an original GIMPshop remake, made by someone who had been using Photoshop for seven years! GIMPshop Reloaded has an interface similiar to Adobe's product, shortcuts, plugin and much more, everything you need to have a real Photoshop feel.

Download
Multiplatform

Easy migration

Photoshop Shortcut

Don't need to learn GIMP Shortcut, or do everything with mouse, just use your keyboard as you already known!

Familiar GUI

No more two-windows or white GUI, everything is where you want to be!

Multiplatform

GIMPshop Reloaded can be installed everywhere GIMP can be run, doesn't matter if you are a win/mac/linux-fag.

Totally malware free!

You can still find various gimpshop(dot)anything on web, but they are scam! The original Gimpshop project is dead and no more supported by developers, so we decided to launch this Reloaded version, totally malware free and open source.

If you don't trust me, you can check source code on Github by your own.
